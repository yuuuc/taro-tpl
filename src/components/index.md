# 组件目录结构

如有与目录结构相关的代码，方法对应的文件下,组件包可自行扩展

```
├─py-xxx-xxx // 某组件
| ├─index.ts // 组件内公共代码
| ├─index.vue // vue文件
| ├─type.ts // 类型定义
| ├─service.ts // 业务逻辑
| ├─index.md // 组件文档
| ├─......
```
